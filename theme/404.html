Frequently Asked Questions

How to watch videos from your browser?

Currently Google Chrome doesn't support vlc player plugin and its build-in video player sucks. We highly recommend you to use mozilla firefox with mozilla-plugin-vlc to watch all the videos in our repo.

GNU/Linux Debain based distros

If you are using Debian based GNU/Linux like Ubuntu, you can install mozilla-plugin-vlc by running the below command in a terminal:

$ sudo apt-get install mozilla-plugin-vlc
